# 腾讯云 Webify 部署指南

## 前提条件
- 有腾讯云账号
- 已登录腾讯云控制台

---

## 方法一：直接上传部署（最快）

### 步骤

1. **进入 Webify 控制台**
   - 打开：https://webify.cloudbase.net/
   - 或从腾讯云控制台 → 云开发 → Webify 进入

2. **创建新应用**
   - 点击「新建应用」
   - 选择「从本地文件夹上传」

3. **上传文件**
   - 上传 `index.html` 文件
   - 或者把整个 `landing-page` 文件夹拖进去

4. **配置域名**
   - 系统会分配一个默认域名（如 `xxx.webify.cloudbase.net`）
   - 可以绑定自定义域名（需要已备案）

5. **完成部署**
   - 点击部署，几秒钟即可完成
   - 访问分配的域名查看效果

---

## 方法二：Git 仓库部署（推荐长期维护）

### 步骤

1. **创建 Git 仓库**
   - 在 GitHub/Gitee 创建新仓库
   - 把 `landing-page` 文件夹内容推送上去

2. **在 Webify 导入**
   - 选择「从 Git 仓库导入」
   - 授权并选择仓库
   - 自动部署

3. **后续更新**
   - 修改代码 → 推送到 Git → Webify 自动更新

---

## 部署后要做的

### 1. 修改联系方式
部署后记得更新 `index.html` 中的：
- 电话/微信
- 具体地址
- 小红书/抖音链接

### 2. 绑定自定义域名（可选）
- 如果有已备案的域名，可以绑定
- 域名更有利于 SEO 和用户记忆

### 3. 提交到搜索引擎
- 百度站长平台提交站点
- 加快搜索引擎收录速度

## 常见问题

**Q: Webify 免费吗？**
A: 有免费额度，静态网站一般够用

**Q: 需要备案吗？**
A: 用 Webify 默认域名不需要，绑定自定义域名需要备案

**Q: 如何更新内容？**
A: 方法一需重新上传；方法二改 Git 代码自动更新
